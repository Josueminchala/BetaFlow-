# Financial Track - AI-Powered Market Intelligence Dashboard
# Setup Requirements and Dependencies

## Project Overview
This is a Next.js 14 TypeScript project with Tailwind CSS for styling.
The dashboard includes portfolio management, news analysis, stock tracking, and AI integration features.

## System Requirements
- Node.js 18.0.0 or higher
- npm 9.0.0 or higher
- Git (for version control)

## Core Dependencies (package.json)
The main dependencies are managed through package.json, not requirements.txt.
Run: npm install

### Production Dependencies:
- next: 14.0.4 (React framework)
- react: ^18 (UI library)
- react-dom: ^18 (React DOM rendering)
- typescript: ^5 (TypeScript support)

### UI Component Libraries:
- @radix-ui/react-*: Various UI primitives (tabs, dialogs, etc.)
- lucide-react: ^0.294.0 (Icon library)
- class-variance-authority: ^0.7.0 (CSS class management)
- clsx: ^2.0.0 (Conditional class names)
- tailwind-merge: ^2.0.0 (Tailwind CSS class merging)

### Development Dependencies:
- @types/node: ^20 (Node.js TypeScript types)
- @types/react: ^18 (React TypeScript types)
- @types/react-dom: ^18 (React DOM TypeScript types)
- eslint: ^8 (Code linting)
- eslint-config-next: 14.0.4 (Next.js ESLint config)
- autoprefixer: ^10.4.21 (CSS vendor prefixes)
- postcss: ^8.5.6 (CSS processor)
- tailwindcss: ^3.4.18 (CSS framework)

## Setup Instructions

### 1. Clone the Repository
```bash
git clone <repository-url>
cd BetaFlow-
```

### 2. Install Dependencies
```bash
npm install
```

### 3. Run Development Server
```bash
npm run dev
```

### 4. Build for Production
```bash
npm run build
npm start
```

## Environment Variables (Future Backend Integration)
Create a .env.local file for API keys:
```
# Gemini AI API
GEMINI_API_KEY=your_gemini_api_key

# News API
NEWS_API_KEY=your_news_api_key

# ElevenLabs API
ELEVENLABS_API_KEY=your_elevenlabs_api_key
```

## Project Structure
```
├── components/          # React components
│   ├── tabs/           # Tab components (Summary, News, Portfolio, etc.)
│   └── ui/             # Reusable UI components
```
